# 🛡️ AI Automated Code Reviewer

A professional Python tool that uses **Bandit** for security scanning and **Ollama (Qwen2.5-Coder)** for AI-driven logic reviews.

## 🚀 Features
- **Syntax Validation:** Catches broken Python code before analysis.
- **Security Scanning:** Detects hardcoded passwords and dangerous shell commands.
- **AI Insights:** Provides detailed feedback on logic and best practices.
- **Web UI:** Built with Streamlit for a smooth user experience.

## 🛠️ Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/automated-code-reviewer.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run src/app.py`
